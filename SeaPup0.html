Several Sea Pups have been built as wood frame non-folders. In fact, any of the boats in this manual can be built as wood or aluminum frame non-folders.

The wood frame/ PVC skin Sea Pup's pictured were built as a neighborhood father / son project headed up by Steve Phillips.

Freeboard - 2.5"+
Prismatic (Cp) .57
Deadrise - 7 deg.
