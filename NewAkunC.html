The wood frame is finished in Watco Danish oil. The non-traditional construction is similar to the wood frame "Sea Rider". The skin will be clear unsupported Vinyl like the Sea Rider. The deck profile aft is a raised flat surface similar to the Sea Tour-R kayaks. This will allow greater storage than a flat gunwale height aft deck and will be useful for entry / exit and rolling. The forward deck is a single stringer Aleut style deckridge. Detailed instructions for both the wood frame "Nikumi" and "Sea Rider" will be added to the manual soon.
